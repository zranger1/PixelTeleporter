PixelTeleporter

A Library by ZRanger1 for the Processing programming environment.
Last update, 09/26/2020.

View pixels from your hardware LED controller on a computer monitor.
PixelTeleporter makes it simple to write sketches that receive pixels from a hardware LED controller, then arrange and render them in 3D on your computer. With this toolset and the included the examples, you can quickly prototype almost any physical arrangement of LEDs.

Installation

Unzip and put the extracted PixelTeleporter folder into the libraries folder of your Processing sketches. Reference and examples are included in the PixelTeleporter folder.

Source. The source code of PixelTeleporter is available at GitHub.
